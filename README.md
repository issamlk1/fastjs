# Fast Js
Fast js is library to make typing long boring js lines easier with only one line of code 
## Getting Started

This plugin have some tools that make typing js more fun and easy to developers

## How its work
- Redirect method

```
//old  
window.location = url;
//new 
redirect(url);
```

- console.log method

```
//old  
console.log(data)
//new 
cl(data);
```
```
//old  
console.error(data)
//new 
ce(data);
```

- inArray method

```
//old  
return arr.indexOf(data) > -1
//new 
inArray(arr,data);
```




### Installing

You can just install it from npm 

npm command 

```
npm i --save fastjs
```

then 

```
require('fastjs');
```
